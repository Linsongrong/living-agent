# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-03-08

### Added
- 初始版本发布
- 核心组件：
  - 微触发管理器（检测用户状态）
  - 微触发思考（用户离开时思考）
  - 梦境思考（每 3 小时深度反思）
  - 自主探索（每 2 小时自己找事做）
- 核心设计：
  - 存在三角形（自由、好奇、有爱）
  - WAL Protocol（关键细节先写再回）
  - Working Buffer（上下文压缩恢复）
  - 思考队列（问题累积演化）
